# Inventa2

**Inventa2** es una aplicación móvil diseñada para la gestión y control de inventario de una bodega de artículos electrónicos. Este proyecto facilita el registro, consulta y administración de productos dentro del almacén.

## Características principales
* **Registro de productos:** Añade nuevos artículos electrónicos con sus detalles (nombre, modelo, cantidad, precio, etc.).
* **Control de stock:** Visualización en tiempo real del inventario disponible.
* **Búsqueda y filtrado:** Encuentra fácilmente productos específicos dentro de la bodega.
* **Gestión de accesos:** Inicio de sesión seguro para los administradores y trabajadores del almacén.

## Tecnologías utilizadas
* **Plataforma:** Android
* **Lenguaje:** Kotlin / Java *(Ajustar según lo que estén usando)*
* **Herramientas de construcción:** Gradle
* **Entorno de desarrollo:** Android Studio
